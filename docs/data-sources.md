# Data Sources

The first visual milestone uses mock data only. This document records future source direction so the prototype does not imply live coverage before ingestion exists.

## MVP

- Static mock country intelligence records.
- Static mock recent incidents.
- No crawlers.
- No third-party API integration.
- No database.

## Future Source Categories

- Government travel advisories.
- Public incident and conflict datasets.
- Reputable news and wire sources.
- Weather, disaster, and health alerts.
- Internal analyst notes, if the product later supports private workspaces.

## Source Principles

- Store source attribution with every ingested record.
- Prefer structured feeds and documented APIs over scraping.
- Track source freshness and confidence.
- Keep automated summaries traceable to source material.
