# MVP Scope

The MVP is the first visual milestone for SafetyMap Intelligence. It should demonstrate the core dashboard experience with mock data only.

## Included

- Web dashboard.
- Interactive map prototype.
- Mock country intelligence data.
- Country search.
- Country side panel.
- Recent incidents/timeline UI.
- Basic responsive layout for desktop-first use.

## Excluded

- Authentication and user accounts.
- Real data ingestion.
- AI analysis, summarization, or recommendations.
- Database persistence.
- Backend API.
- Crawlers or scrapers.
- Kubernetes.

## Acceptance Criteria

- A user can open the dashboard and see countries represented on an interactive map.
- A user can search for a country from mock data.
- Selecting a country opens a side panel with mock risk context and recent incidents.
- The timeline UI is understandable without relying on live data.
- The implementation remains frontend-only for this milestone.
